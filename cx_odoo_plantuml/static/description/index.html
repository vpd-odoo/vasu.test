HOW IT WORKS

  1. Select module and press "Save"
  2. Download file(s)
  3. Upload file or paste PlanUML code to any PlanUML editor
  4. Useful links (external links are not clickable from App Store):
    • LiveUML online viewer: https://liveuml.com/
    • PlantUML online viewer: http://www.plantuml.com/plantuml
    • JetBrain IDE plugins: https://plugins.jetbrains.com/search?search=plantuml
    • VSCode extensions: https://marketplace.visualstudio.com/search?term=plantuml&target=VSCode&category=All%20categories&sortBy=Relevance
  5. It is strongly recommended that you use IDE or command line renderer to render diagrams containing fields
    For command line rendering:
    • Download plantuml.jar from https://plantuml.com/download
    • Check command line parameters here: https://plantuml.com/command-line
    • This is and example of the command that renders diagram into png image: java -DPLANTUML_LIMIT_SIZE=16384 -jar plantuml.jar test.puml
      Note: PLANTUML_LIMIT_SIZE=16384 parameter is used to increase maximum image size
